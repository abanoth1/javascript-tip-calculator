# JavaScript Tip Calculator

This project is a simple front-end web application that calculates a tip amount based on user input. 
It demonstrates core JavaScript concepts such as DOM manipulation, event handling, and basic application logic using clean HTML and CSS.

## Features
- User input for bill amount
- Tip percentage selection
- Dynamic calculation of tip and total amount
- Instant updates without page reload
- Clean and responsive user interface

## Skills demonstrated
- JavaScript (ES6 basics)
- DOM manipulation
- Event handling
- Form validation
- HTML & CSS fundamentals
- Front-end application logic
